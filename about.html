ABOUT THIS SITE

Just what is this site about, anyways?

Monday, December 5

Hello! This is Joey Kim's web site (read: blog)! On this site, I will post anything that interests me or any projects I happen to do.

This site is unique in that it uses a lot of JavaScript templating code. It simplifies the maintenence of this site significantly, and it helps keep the files small. Therefore, a small change in one file is reflected in all the files, instead of having to tediously change each file one by one. As a consequence, you need to look at this site with a HTTP server.

A custom "search service" built by Joey has been implemented for this site specifically, although it can be used elsewhere. The setup is easy, and that will explain the <!-- BEGIN_SEARCH --> comments in the HTML source.

Most of the code is written in JavaScript and HTML. I would like to stress that everything on the site is in the public domain, so anyone can copy-paste from it! (However, I would like to stress that I would like it if you credited me for my hard work).
